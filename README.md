# Image Gallery

## Description
This project is a responsive **Image Gallery** developed using **HTML5** and **CSS3**. It displays a collection of images in a clean and organized layout with a modern design. The gallery is fully responsive, providing an excellent viewing experience across desktop, tablet, and mobile devices.

## Features
- Responsive and mobile-friendly design
- Clean and modern gallery layout
- Organized image grid
- Hover effects on images
- Fast and simple navigation
- Built using HTML5 and CSS3

## Technologies Used
- HTML5
- CSS3

## Project Structure
```
project/
│── gallery.html
│── style.css
└── README.md
```

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `gallery.html` in any modern web browser.

## Outcome
This project demonstrates the creation of a responsive image gallery using HTML5 and CSS3 with a clean layout and modern design.
